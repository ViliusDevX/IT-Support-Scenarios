# Ticket: Unable to access internal resources by name

## Ticket Details

**Ticket ID:** IT-004

**Priority:** Medium

**Category:** Network / DNS Issue

**Affected device:** WIN11-CLIENT01

**Affected resource:** Internal domain resources

**Reported by:** Helpdesk / User Support

**Status:** Open

## Original Request

Hello,

A user is reporting that they are having trouble accessing internal company resources from their workstation.

The computer is connected to the network and internet access appears to be working, but some internal resources do not open when the user tries to access them by name.

The user said they were able to access the same resources previously, but now Windows either cannot find the resource or takes a long time before failing.

Could someone please check why this workstation is having trouble reaching internal resources by name?

## Reported Symptoms

* Device name: `WIN11-CLIENT01`
* User can sign in to Windows normally
* Internet access appears to be working
* Internal resources fail when accessed by name
* Issue appears limited to this workstation
* Other users have not reported the same problem so far

## Business Impact

The user cannot reliably access internal domain resources needed for normal work.

## Additional Notes

This may be a network or name resolution issue. Please verify whether the workstation is receiving the correct network configuration and can resolve internal domain names properly.
