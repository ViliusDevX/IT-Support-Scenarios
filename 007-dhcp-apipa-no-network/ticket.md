# Ticket: Workstation has no network access

## Ticket Details

**Ticket ID:** IT-007

**Priority:** Medium

**Category:** DHCP / Networking

**Affected device:** WIN11-CLIENT01

**Affected resource:** Network connectivity

**Reported by:** Helpdesk / User Support

**Status:** Open

## Original Request

Hello,

A user is reporting that their workstation has no network access after signing in this morning.

The user can log in to Windows, but they cannot open internal shared folders, access internal resources, or browse the internet. The network icon shows a connectivity problem.

Other users appear to be working normally, so this may be limited to this workstation.

Could someone please check why this device is not getting network access?

## Reported Symptoms

* Device name: `WIN11-CLIENT01`
* User can sign in to Windows
* Internet access is not working
* Internal resources are not reachable
* Shared folders cannot be opened
* Issue appears limited to this workstation
* Other workstations appear to have normal network access

## Business Impact

The user cannot access internal resources, shared files, or internet-based tools needed for normal work.

## Additional Notes

This may be related to the workstation's network configuration. Please verify whether the device is receiving a valid IP address, gateway, and DNS settings.
