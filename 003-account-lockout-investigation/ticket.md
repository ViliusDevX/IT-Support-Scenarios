# Ticket: User account keeps locking out

## Ticket Details

**Ticket ID:** IT-003

**Priority:** Medium

**Category:** Account / Authentication Issue

**Affected user:** `vilius\it003.user`

**Affected device:** WIN11-CLIENT01

**Reported by:** Helpdesk / User Support

**Status:** Open

## Original Request

Hello,

A user is reporting that their domain account keeps getting locked out during the workday.

The account was unlocked earlier, and the user was able to sign in again, but after a short time they were locked out once more. The user says this has happened more than once today.

They are currently unable to work reliably because they keep losing access to their Windows session and domain resources.

Could someone please check why this account keeps locking out?

## Reported Symptoms

* User account: `vilius\it003.user`
* Device name: `WIN11-CLIENT01`
* User can sign in after the account is unlocked
* Account locks again after some time
* User cannot reliably access domain resources while the issue is happening
* No other users have reported the same issue so far

## Business Impact

The user is repeatedly losing access to their workstation and domain resources, which is interrupting normal work.

## Additional Notes

The issue appears to affect one user account rather than the whole domain. Please investigate the source of the repeated lockouts and confirm whether the user account, workstation, or another authentication source is causing the problem.
